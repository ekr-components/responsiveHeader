# responsiveHeader

## How to use this component

```html
<div data-component="header">
	<header data-header-responsive="desktop">
		<!-- Header and menu goes here -->
	</header>
	<header class="header-mobile" data-header-responsive="mobile">
		<!-- Header and logo go here -->
		<span data-header-mobile="trigger"></span>
		<div data-header-mobile="menu">
			<!-- Menu contents go here -->
		</div>
	</header>
</div>
```
